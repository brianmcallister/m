Basic example.

This basic example uses all the default settings, without any of the included CSS themes. This is what your dropdowns will look like if you don't use any CSS at all. The whole point of this plugin is that you want to write your own CSS from scratch to fit in your project.

Monkeypatched jQuery#val

You have the option to use a build of this script that uses a monkeypatched val() function from jQuery. This monkeypatch adds a event trigger when setting a value on your select box, so you don't need to do anything special to update m. If you choose not to use the monkeypatch, it's just a matter of firing a change event on the select box after setting the value.
